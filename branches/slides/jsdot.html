API

Changing graphs through the API.

 
	1:	var j = new JSDot();
	2:	var g = j.getGraph();
	3:	var n = g.createNode();
	4:	n.setPosition([100, 80]);
	5:	n.setLabel('box node');
	6:	n.setStencil('box');
